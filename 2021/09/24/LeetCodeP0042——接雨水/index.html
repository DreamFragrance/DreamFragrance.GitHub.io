LeetCodeP0042——接雨水

发表于 2021-09-24 分类于 LeetCode 阅读次数： Valine：

题目

给定 n 个非负整数表示每个宽度为 1 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

示例1

示例2

提示：

  • n == height.length
  • 0 <= n <= 3 * 104
  • 0 <= height[i] <= 105


题解1: 动态规划

求出在这个点从左边和右边遍历的最高值，取他们之间的较小值，即两个端点的较小值，减去该点的高度就是这个点可以储水的量了。

用时：1ms

时间复杂度：O（n）

空间复杂度：S（n）


题解2: 双指针

与动态规划类似，但采用双指针可以省去dp数组。

只移动短的那一头，因为短的那一头左右都已经有了边界，可以直接计算出储水量。

用时：0ms

时间复杂度：O（n）

空间复杂度：S（1）
